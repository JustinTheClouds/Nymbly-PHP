Plugins

Configs

Private configs defined at runtime
Editable configs defined in the application config file

CSS and JS Files

By default Nymbly will try to load all styles and scripts set in your plugins configs styles and scripts settings.

Hooks

Actions

Actions are hooks that occur at a specific time during the applications execution. Use action hooks to execute code at the specified time.

Filters

Filters are hooks that allow you to modify data. All filters hook are passed in a parameter that you can modify as needed. Then your hook must return the modified parameter. If the parameter is not returned it will cause unexpected results and may cause errors.
